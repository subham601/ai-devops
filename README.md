# Ecommerce Web Application (FastAPI + Vanilla JS)

This repo contains a simple ecommerce demo:
- FastAPI backend with JWT login
- Products catalog + add-to-cart via REST API
- Vanilla HTML/CSS/JS frontend
- Docker support

Run locally (after implementation):
```bash
cp .env.example .env
docker compose up --build
```

Then open:
- http://localhost:8000
